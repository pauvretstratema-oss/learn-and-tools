Configurer VS Code pour débutants

Visual Studio Code (VS Code) est l’un des éditeurs de code les plus utilisés aujourd’hui. Il est léger, gratuit, très puissant, et fonctionne très bien avec Python… mais aussi avec plein d’autres langages.

Objectif : installer VS Code, ajouter les bonnes extensions et le configurer pour coder confortablement.

1. Installer Visual Studio Code

Télécharge VS Code depuis le site officiel : code.visualstudio.com.

Pendant l’installation, tu peux laisser les options par défaut. Sur Windows, coche l’option qui permet d’ajouter “Code” au menu clic droit, c’est pratique.

2. Installer l’extension Python

Pour bien travailler avec Python, installe l’extension officielle :

  • Ouvre VS Code
  • Va dans l’onglet Extensions (icône carrés à gauche)
  • Cherche Python (par Microsoft)
  • Clique sur Install

Cette extension ajoute :

  • coloration syntaxique améliorée
  • exécution facile des scripts Python
  • débogage (breakpoints, variables, etc.)
  • suggestions intelligentes (IntelliSense)

3. Ouvrir un dossier de projet

L’une des bonnes habitudes avec VS Code, c’est de travailler dans un dossier de projet :

  • Crée un dossier, par exemple projets_python
  • Place dedans un fichier comme test.py
  • Dans VS Code, va dans File → Open Folder... et choisis ce dossier

À partir de là, VS Code reconnaît ton dossier comme un projet : explorateur de fichiers à gauche, scripts listés, terminal intégré, etc.

4. Lancer un script Python depuis VS Code

Crée un fichier hello.py avec le contenu suivant : 

print("Hello depuis VS Code !")

Tu as plusieurs façons de l’exécuter :

  • Clic droit dans l’éditeur → Run Python File in Terminal
  • Ou ouvrir un terminal intégré (View → Terminal) puis :
python hello.py
Le terminal intégré est très pratique : tu peux lancer des commandes sans quitter VS Code.

5. Extensions utiles pour bien débuter

Voici quelques extensions simples et utiles :

  • Python (Microsoft) – indispensable pour Python
  • Pylance – meilleures suggestions et analyse de code
  • Code Runner – exécution rapide de petits bouts de code
  • Material Icon Theme – icônes de fichiers plus jolies
  • GitHub Theme ou autre thème sombre – pour le confort des yeux

6. Personnaliser le thème et la police

Un environnement agréable donne envie de coder plus longtemps. Tu peux changer :

  • le thème : File → Preferences → Color Theme
  • la police et la taille : File → Preferences → Settings, cherche “Font”

Par exemple, tu peux augmenter la taille de la police à 14 ou 16 si tu trouves le texte trop petit.

7. Raccourcis clavier de base

Quelques raccourcis très utiles :

  • Ctrl + S : sauvegarder le fichier
  • Ctrl + ` : ouvrir / fermer le terminal intégré
  • Ctrl + B : afficher / cacher l’explorateur de fichiers
  • Ctrl + P : ouvrir rapidement un fichier par son nom
  • Ctrl + / : commenter / décommenter la ligne
Au début, ne cherche pas à tout retenir. Commence avec 2–3 raccourcis que tu réutilises souvent, et le reste viendra naturellement.

Conclusion

Avec VS Code correctement installé et configuré, tu disposes d’un environnement de développement moderne pour Python (et bien d’autres langages).

Tu as maintenant :

  • un éditeur puissant
  • l’extension Python installée
  • la possibilité d’exécuter tes scripts directement
  • un terminal intégré sous la main

Prochaine étape logique : continuer avec les bases de Python (variables, conditions, boucles…), ou configurer un premier petit projet concret (script utilitaire, mini-jeu, etc.).
