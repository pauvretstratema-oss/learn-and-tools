Démarrer avec Python

Python est l’un des langages les plus simples à apprendre. Il est utilisé pour automatiser des tâches, développer des applications, créer des jeux, manipuler des données et même faire de l’IA.

Objectif : Installer Python, écrire ton premier script, et comprendre les bases.

1. Installer Python

Pour installer Python, rends-toi sur le site officiel : python.org/downloads.

Après installation, vérifie que Python est bien disponible en ouvrant un terminal : 

python --version

Tu devrais voir quelque chose comme :

Python 3.12.0

2. Ton premier programme Python

Ouvre un éditeur de texte ou Visual Studio Code, crée un fichier appelé test.py et écris : 

print("Bonjour, Python !")

Exécute-le :

python test.py

Tu devrais voir : 

Bonjour, Python !

3. Variables : comprendre le principe

Une variable est simplement une boîte dans laquelle tu stockes une information. Exemple : 

age = 21
nom = "Alex"
print("Nom :", nom)
print("Âge :", age)

Sortie :

Nom : Alex
Âge : 21

4. Entrées clavier : demander quelque chose à l’utilisateur

Python peut lire ce que tape l’utilisateur :

prenom = input("Quel est ton prénom ? ")
print("Bonjour", prenom)

Astuce : input() renvoie toujours du texte (str).

5. Faire un petit programme utile

Voici un exemple simple : une addition automatique.

a = int(input("Nombre 1 : "))
b = int(input("Nombre 2 : "))
print("Somme =", a + b)

Pas mal pour un début, non ? Tu sais déjà installer Python, exécuter un script, créer des variables et faire tes premiers programmes.

Conclusion

Ce premier article pose les bases. Dans les suivants, tu apprendras :

  • les conditions (if / else)
  • les boucles
  • les fonctions
  • les listes, dictionnaires…
  • comment créer un vrai petit projet
